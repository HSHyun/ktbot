[Task]

3. Keep all numbers, dates, percentages, and proper nouns exactly as in input.

5. If comments are irrelevant or noisy, ignore them.
6. If attached images add factual context, reflect that briefly.
7. Do not invent facts that are not in the input.

- Line 1: write a new concise Korean title summarizing the post (no prefix).
- Do not start Line 1 with "-", "*", numbers, or any bullet marker.
- Line 2+: 2-3 Korean sentences summary (no bullets, no labels).
- Output only the result text.
